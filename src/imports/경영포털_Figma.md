## 3. 공통 컴포넌트 명세 ✅

### 3-1. Header

| 요소 | 타입 | 내용 |
|------|------|------|
| 로고 | Image | 포털 로고 |
| 담당 토글 | Toggle Button Group | 중부 / 서부 — 선택 시 전체 테마 전환 |
| 기준일시 | Text | "기준일시 : YYYY-MM-DD HH:mm" |
| 업데이트 버튼 | Button / Primary | 클릭 시 로딩 오버레이 + 전체 데이터 갱신 |

### 3-2. 담당별 테마 토큰

| 토큰 | 중부 | 서부 |
|------|------|------|
| `--region-primary` | #3617CE | #00A86B |
| `--region-dark` | #270FA8 | #007A4D |
| `--region-light` | #EEF0FF | #E6F7F1 |
| `--region-border` | #D8DBFF | #B3EDD8 |

### 3-3. 공통 컬러 토큰

| 토큰 | 값 | 용도 |
|------|-----|------|
| `--ok` | #00A86B | 정상 |
| `--warn` | #FF7A00 | 점검 필요 |
| `--danger` | #EA002C | 긴급 |
| `--gray-900` | #1A1A1A | 본문 텍스트 |
| `--gray-700` | #3D3D3D | 서브 텍스트 |
| `--gray-500` | #767676 | 보조 텍스트 |
| `--gray-300` | #C4C4C4 | 비활성/보더 |
| `--gray-100` | #F5F5F5 | 배경 구분 |
| `--gray-50` | #FAFAFA | 카드 배경 |

### 3-4. KPI 카드

| Prop | Type | 설명 |
|------|------|------|
| label | string | 카드 제목 |
| value | number | 주요 수치 |
| unit | string | 단위 |
| yoy | number \| null | 전년 대비 |
| variant | 'default' \| 'warn' \| 'danger' | 강조 색상 |

### 3-5. 드릴다운 사이드바

| 속성 | 내용 |
|------|------|
| 형태 | Overlay (기존 화면 유지, 우측 슬라이드 인) |
| 너비 | 380px |
| 오버레이 | rgba(26,26,26,0.45) |
| 닫기 | 오버레이 클릭 또는 ✕ 버튼 |

### 3-6. 테이블 필터 공통 구조

| 요소 | 설명 |
|------|------|
| 키워드 검색 | 명칭/주소 실시간 검색 + 하이라이트 |
| 상태 Multi-Tag | 전체/정상/점검필요/긴급 복수 선택 |
| 컬럼 헤더 필터 | 체크박스 드롭다운 |
| 필터 칩 | 활성 필터 인라인 표시 + 개별 제거 |
| 결과 카운트 | "총 N개 중 M개 표시" |
| Empty State | 결과 없음 UI + 초기화 버튼 |

## 5. 현황 (Status) — 시설

### 5-1. 통합국 ✅ 설계완료

```
┌─────────────────────────────────────────────────────────┐
│  1차 탭 : 현황 / 2차 탭 : 시설 / 3차 탭 : [통합국]      │
├──────────┬──────────┬──────────┬────────────────────────┤
│ KPI:     │ KPI:     │ KPI:     │ KPI:                   │
│ 총 국사  │ 발전기   │ 발전기   │ 긴급 처리              │
│ 수       │ 설치     │ 미설치   │ 건수                   │
├──────────┴──────────┼──────────┴────────────────────────┤
│  지도 + 핀 (6col)   │  임차형태 도넛 + 발전기 도넛       │
├─────────────────────┴───────────────────────────────────┤
│  국사 세부 현황 테이블                                   │
│  컬럼: 국사명 | 지역 | 임차형태 | 발전기 | 수전용량 |    │
│        국사등급 | 축전지 | Moderniza | 상태 | 상세       │
│  필터: 키워드 + 상태태그 + 컬럼헤더 필터 + 칩           │
└─────────────────────────────────────────────────────────┘
```

**드릴다운 사이드바 탭**

| 탭 | 콘텐츠 |
|----|--------|
| 기본정보 | 위치/임차/등급/설비요약/현재알림 |
| 설비현황 | 월별 가동률 Bar Chart + 설비 목록 |
| 점검이력 | 타임라인 (최근 6건) + 점검 통계 |
| Moderniza | 진행률 프로그레스 바 + 단계별 현황 |

**API 연동**

| Endpoint | 설명 |
|----------|------|
| GET `/api/v1/tonghabguk/{region}/kpi` | KPI 카드 |
| GET `/api/v1/tonghabguk/{region}/table` | 테이블 목록 |
| GET `/api/v1/tonghabguk/{region}/pins` | 지도 핀 |
| GET `/api/v1/tonghabguk/{region}/lease` | 임차형태 집계 |
| GET `/api/v1/tonghabguk/{region}/detail/{id}` | 단건 상세 |
